# Sentiment Analysis Project

## Overview
This project performs sentiment analysis on textual review data using Python and TextBlob.

## Tools Used
- Python
- Pandas
- TextBlob
- Matplotlib
- Jupyter Notebook

## Features
- Text preprocessing
- Sentiment classification
- Positive/Negative/Neutral analysis
- Sentiment visualization

## Workflow
1. Load review dataset
2. Analyze sentiment polarity
3. Classify reviews
4. Visualize sentiment distribution

## Outcome
The project demonstrates basic Natural Language Processing (NLP) and sentiment analysis concepts using textual datasets.
